# 💌 Wedding Invitation — Hashim & Shamshad 💍✨

A beautifully designed, modern, and fully responsive **Digital Wedding Invitation Website** created to celebrate the special occasion of **Hashim & Shamshad** 💖

This project showcases creativity, UI/UX design skills, and frontend development expertise using modern web technologies.

---

## 🧑‍💻 About This Project

This is a **digital wedding invitation card** designed to provide an elegant and interactive experience for guests 🎉

It replaces traditional invitation cards with a modern web-based solution that includes animations, styling, and responsive design for all devices 📱💻

---

## 🌟 Project Purpose

The goal of this project is to:

* Create a **visually appealing wedding invitation**
* Provide a **modern digital experience**
* Showcase **frontend development skills**
* Make invitations easily shareable via link 🌐

---

## 🧭 Website Sections

The invitation website includes:

* 💌 **Welcome Section** – Couple introduction (Hashim & Shamshad)
* 📅 **Event Details** – Wedding date, time & venue
* 📖 **Story Section** – Journey of the couple
* 🖼️ **Gallery** – Beautiful images & memories
* 📍 **Location Map** – Easy navigation to venue
* 📬 **Contact / RSVP** – Guest response section

## 📦 Installation & Setup

```bash
# Clone repository
git clone https://github.com/Arbazkhan8957/Invitation-card.git

# Go to project folder
cd Invitation-card

# Install dependencies
npm install

# Run project
npm run dev
```

---

## 🚀 Deployment

This project is deployed using **GitHub Pages**

```bash
npm run deploy
```
